Portfolio Builder Website

For my next large personal side project, I will be building a resume/portolio website builder that will allow users to sign up to the service, choose from a couple of templates, use custom themes, upload their information and have a ready to go online portfolio.

This project is ambitious, and I expect it to take many months to complete. I will be looking to delve more deeply in to UX/UI design on both the templates - so the end product for the user is visually appealing and easy to use - and the GUI for user uploads (so the experience of building their portfolio is enjoyable for the user).

It will be important that I read up on and implement security procedures to help keep user's information secure.

Finally, while developing, I must be mindful of writing readable and scaleable code so that I can potencially publish this site for a wider audience in future.
